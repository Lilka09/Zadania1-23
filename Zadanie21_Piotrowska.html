Pojęcie Opis
Co znaczy skrót ID? ID to unikalny identyfikator elementu na stronie, który pozwala na jego jednoznaczną identyfikację i stylizację.
Jaki jest cel stosowania ID? ID służy do identyfikacji pojedynczego, unikalnego elementu na stronie i nadawania mu stylów lub manipulacji za pomocą JavaScript.
Jakie jest znaczenie znaku #? Znak # używany jest do wywoływania identyfikatora ID w CSS.
Jak następuje wywołanie ID? Wywołanie ID w CSS odbywa się za pomocą znaku #, np. #mojeID { color: red; }. W HTML przypisanie ID odbywa się za pomocą id="mojeID".
Uwagi na temat stosowania ID
  • ID może wystąpić tylko raz na stronie.
  • W przypadku powielania ID mogą wystąpić nieprzewidziane konflikty.
  • Jeśli istnieje potrzeba wielokrotnego użycia, lepiej zastosować klasę (class).
